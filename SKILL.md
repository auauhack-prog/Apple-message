# 苹果短信自动化 V1.0

## 这是什么

一个自动模拟苹果手机发送短信的工具。把序列号导入后，系统会自动分配设备、发送短信、记录发送时间，如果苹果封禁了你的序列号会弹出提醒。

---

## 安装后第一步：配置

在项目文件夹里找到 `.env.example`，复制一份改名为 `.env`，填入你的腾讯云密钥：

```
TENCENT_SECRET_ID=你的SecretId
TENCENT_SECRET_KEY=你的SecretKey
USER_PHONE=你的手机号
SMS_CONTENT=要发送的短信内容
```

> 腾讯云密钥获取地址：https://console.cloud.tencent.com/cam/capi

---

## 使用步骤

### 1. 准备序列号文件

在项目文件夹里新建或编辑 `serials.txt`，一行一个序列号：

```
ABC123456789
DEF987654321
GHI111222333
```

### 2. 启动

双击 `启动.command`

### 3. 系统自动执行

```
登录 → 导入序列号 → 去重检查 → 分配虚拟设备
    → 模拟发送 → 记录发送时间
    → 生成记录(records.json) → 封禁检测 → 告警/继续
```

### 4. 查看结果

- 发送记录：`records.json`
- 被封禁提示：终端窗口

## 常见问题

**Q: 序列号从哪里来？**
A: 苹果设备的序列号，在「设置 → 关于本机」查看。

**Q: 重复序列号怎么办？**
A: 自动去重，最多重试 3 次后跳过。

**Q: 短信真的会发出去吗？**
A: 当前版本模拟发送，接入真实 API 后才会实际发送。

**Q: 序列号被封禁了？**
A: 每次发送后自动检测，被封禁会弹告警。
